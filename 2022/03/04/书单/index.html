一枚书单

发表于 2022-03-04 更新于 2021-09-17 分类于 杂记 ， 碎碎念
唯有晨光从容，没有遗憾，新鲜如初

学弟让我推荐书单，刚好整理一下，顺便放上来吧。

我的开发书单

  1. 《java核心技术 卷1》 ：补java基础知识的
  2. 《mysql必知必会》 ： 一本很薄的，mysql语法的工具书
  3. 《高性能mysql》 ：MySQL的中高级教程
  4. 《深入理解java虚拟机》 ：jvm入门读物
  5. 《Java并发编程实战》 ：讲java线程和并发的。这本我买了，但是别人说这本书不适合面试来读（狗头，真正的面向工作面试学习应该读下面这本
  6. 《java并发编程艺术》 ：这本读了电子版，还行，也没那么好。
  7. 《深入理解计算机系统》 ：讲操作系统的。超大超厚，买了吃灰了没读过。
  8. 《算法 （第4版）》 ：讲算法和数据结构的。应该挺好的，上学期看别人读过，另一个学长也推荐过
  9. 《spring揭秘》 ：之前不了解spring框架就硬读，读不懂，不如看看视频或者做一个项目了解spring框架后再读。
  10. 《现代操作系统》 ：最近在读这本，刚开始读感觉还行
  11. 《图解TCP/IP》 ：讲计算机网络的。有人推荐过，不了解，也比较经典

别人的书单

算法与数据结构：

  1. 数据结构（严蔚敏）/大话数据结构 //如果觉得教材无聊就可以看大话系列，印象中里面还有很多诗

  2. 剑指Offer/程序员面试金典/编程珠玑/编程之美/牛客网+leetcode
    程序员笔试面试最优解（左程云）/不如直接看左神的笔试面试指南视频

  3. Java的版本（不是很推荐）：

    • 数据结构与算法经典问题解析（Java语言描述）
    • 图解数据结构（使用Java）

计算机网络 ：

  • 计算机网络（谢希仁）
  • TCP/IP 详解
  • HTTP权威指南
  • 图解TCP/IP
  • 图解HTTP

数据库 ：//数据库主要是多用，书上主要看索引和性能的部分

  • 高性能MySQL/深入浅出MySQL

操作系统 ：

  1. OS原理：

    • 操作系统（课本，黑色的那个）

  2. Linux：

    • Linux私房菜 //鸟哥写的，很全，包括bash部分

    • 跟阿铭学Linux //主要偏重于命令和操作，比较浅显

java：

  • Java疯狂讲义/Java编程思想/Java核心技术 卷1

  • 深入理解Java虚拟机

  • 并发编程的艺术/多线程编程核心技术

  • Effective Java

  • Java程序员面试笔试宝典 //何昊的那本，个人感觉是突击知识点的神器

  • Java程序性能优化

  • 实战Java高并发程序设计

Java Web：

  • Spring实战/轻量级JavaEE 企业应用（红皮，讲SSH的） //主要看最后一部分Spring的就可以

  • 深入JavaWeb技术内幕（阿里 许令波）//这个讲的还是比较深的

  • SpringBoot实战/深入实践SpringBoot

设计模式：

  • 大话设计模式 //通俗易懂

  • 各类博客的总结

分布式与大数据：

  • 分布式服务框架原理与实践

  • 大型网站技术架构

  • Hadoop实战（hadoop体系包括得很全）

其他：

  1. Git：

    • Git权威指南
    • Git官方讲解视频（牛客网有带字幕的）

  2. Redis：

    • Redis实战

还买了docker、springCloud等等一些工具书，因为太小众就不列举了
